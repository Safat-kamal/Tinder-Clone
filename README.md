
## Tinder Clone
 
Tinder clone is a Dating App built using MERN Stacks.
 



## Front End 

- React Js

- React Tinder Card

- Css + Material UI

- Axios 

- Heroku


## Backend

- Express Js

- MongoDB Atlas

- Mongoose

- Heroku
